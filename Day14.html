CallBack Function

A callback function is a functions passed into another function as a argument

        function second (callback){
            console.log("second execution")
            callback()
        }
        function first(){
            console.log('first function')
        }
        first()

CallBack Hell

callback hell is an anti-pattern with multiple nested callbacks which makes code hard to read and debug when dealing with asynchronous logic.

            setTimeout(()=>{
                console.log("1")
                 setTimeout(()=>{
                     console.log("2")
                      setTimeout(()=>{
                          console.log("3")
                           setTimeout(()=>{
                               console.log("4")
                           },2000)
                      },4000)
                 },5000)
            },3000)

Promises

A promise is an a object which representing the eventual completion or failure of an asynchronous operation it gives us to write cleaner code to reduce callback hell.

            function Number(N){
                return new promise((resolve,reject)=>{
                    if(N>10){
                        console.log("greater")
                        resolve();
                    }
                    else{
                        console.log("not greater")
                        reject();
                    }
                })
            }
            Number(20)
            .then((result)=>{
                console.log(result)
            })
            .catch((error)=>{
                console.log(error)
            })

Promise Chaining

Process of executing a sequence of asynchronous tasks one after another using promises is known as promise chaining

        function Number(time,value){
            return new promise((resolve,reject)=>{
                setTimeout(()=>{
                    console.log(value)
                    resolve()
                },time)
            })
        }
        Number(5000,1)
        .then(()=> Number(4000,2))
        .then(()=> Number(3000,4))
        .then(()=> Number(1000,6))
        .then(()=> Number(5000,8))

Async and Await

Async/Await is the extension of promises which we get as a support in the language. You can refer Promises in Javascript to know more about it. Async: It simply allows us to write promises based code as if it was synchronous and it checks that we are not breaking the execution thread.

        async function print(){
            await Number(4000,1)
            await Number(6000,2)
            await Number(3000,3)
            await Number(4000,4)
        }
