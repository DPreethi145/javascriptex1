JavaScript Operators

  • Arithmetic Operators
  • Comparison Operators
  • Bitwise Operators
  • Logical Operators
  • Assignment Operators

Arithmetic Operators

  1. + => Addition Ex: 10+20 = 30
  2. - => Subtraction Ex: 20-10 = 10
  3. * => Multiplication Ex: 10*20 = 200
  4. / => Division Ex: 20/10 = 2
  5. % => Modulus (Remainder) Ex: 20%10 = 0
  6. ++ => Increment Ex: var a=10; a++; Now a = 11
  7. -- => Decrement Ex: var a=10; a--; Now a = 9

Comparison Operators

  1. == => Is equal to Ex: 10==20 = false
  2. === =>Identical (equal and of same type) Ex: 10==20 = false
  3. != => Not equal to Ex: 10!=20 = true
  4. !== => Not Identical Ex: 20!==20 = false
  5. > => Greater than Ex: 20>10 = true
  6. >= => Greater than or equal to Ex: 20>=10 = true
  7. < => Less than Ex: 20<10 = false
  8. < = =>Less than or equal to Ex: 20<=10 = false

Bitwise Operators

  1. & Bitwise AND (10==20 & 20==33) = false
  2. | Bitwise OR (10==20 | 20==33) = false
  3. ^ Bitwise XOR (10==20 ^ 20==33) = false
  4. ~ Bitwise NOT (~10) = -10
  5. << Bitwise Left Shift (10<<2) = 40
  6. >> Bitwise Right Shift (10>>2) = 2
  7. >>> Bitwise Right Shift with Zero (10>>>2) = 2

Logical Operators

  1. " && " Logical AND (10==20 && 20==33) = false
  2. "|| " Logical OR (10==20 || 20==33) = false
  3. "! " Logical Not !(10==20) = true

Assignment Operators

  1. "=" Assign 10+10 = 20
  2. "==" Add and assign var a=10; a+=20; Now a = 30
  3. "-=" Subtract and assign var a=20; a-=10; Now a = 10
  4. "*= " Multiply and assign var a=10; a*=20; Now a = 200
  5. " /=" Divide and assign var a=10; a/=2; Now a = 5
  6. " %=" Modulus and assign var a=10; a%=2; Now a = 0
