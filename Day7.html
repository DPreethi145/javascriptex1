Arrays

Stores Multiples Elements in a Single Variable

Map

Map method will checks all the elements in an array it gives the completly new array as a output

        EX: var arr = [1,2,3,4,5]
            var arr2 = arr.map((a)=>{a=a*2})
            console.log(arr2)

            Output : [2,4,6,8,10]

Filter

Filter method will filter all the elements in an array it returns subset of an array as a output

        EX : var arr = [1,2,3,4,5]
             var arr2 = arr.filter((a)=>{a=a<5})
             console.log(arr2)

             output : [1,2,3,4]

Reduce

Reduce method will reduces all the elements in an array gives single Value as a output

        EX: var arr = [1,2,3]
            var arr2 = arr.reduce((sum,a)=>sum+a,0)
            console.log(arr2)

            output : 6

Push

my using push it push the element at the end of an array

EX : arr.push(10)

Pop

my using pop it removes the element at the end of an array

EX : arr.pop()

Shift

By using shift it Removes the element starting of an array

EX : arr.shift()

UnShift

By using shift it add the element starting of an array

EX : arr.Unshift()

Some and Every

Both these are take in a condition and check if all the elements in th array is satisfying the condition or not, they always return a boolean value.

Ex: some syntax - const val2 = arr.some(x => x%2 == 0);

EX : every syntax - const val3 = arr.every(x => x%2 == 0);
