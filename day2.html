VAR vs LET vs CONST

=> VAR

var is a Global or Functional Scoped.it can be reassigned or redeclared and it can be modified later.

=> LET

let is a Block Scoped.it cannot be reassigned or redeclare. let can be modified later.

=> CONST

const is a Block Scoped.it cannot be reassigned or redeclare.const cannot be modified later.


Hoisting

Hoisting is a default Javascript Mechanism where our variables and Functions declarations are moved to top of their Scope before code execution (),only declarations are hoisted not initializations.

Temporal Dead Zone

The time between the Variables hosted and the variables get initialized is called the Temporal Dead Zone Let and const are also hoisted and are in Temporal Dead Zone
